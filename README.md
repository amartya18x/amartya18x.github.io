# personal website

This site is rebuilt with Astro and keeps the same content as the previous Jekyll version, with a cleaner, minimal layout.

## development

```bash
npm install
npm run dev
```

## build

```bash
npm run build
```

## content locations

- Homepage layout: `src/pages/index.astro`
- Posts (talks/news): `src/content/posts/`
- Publications (single BibTeX file): `src/content/publications.bib`
- Pages (funding items): `src/content/pages/`
- Static assets (images, CV, favicon): `public/`
